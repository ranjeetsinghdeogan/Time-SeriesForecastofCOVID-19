# Time-Series Forecast of COVID-19 Cases (Power BI)

## Project Overview

This project focuses on analyzing and forecasting COVID-19 case trends using time-series data. The objective was to study case patterns for a selected country and predict future case trends using forecasting techniques in Power BI.

## What I Did

* Imported COVID-19 dataset into Power BI
* Cleaned data by removing unnecessary columns
* Filtered the dataset to analyze one specific country
* Calculated a 7-day rolling average of new cases
* Created measures for cumulative vaccinations
* Applied forecasting to predict new cases for the next 30 days
* Calculated month-over-month percentage change in new cases
* Presented insights on trend changes and forecast reliability

## How I Did It

* Used Power BI and Power Query for data preparation
* Created DAX measures for rolling averages and percentage change
* Used line charts to visualize daily case trends
* Applied the **Forecast** feature to generate future predictions
* Adjusted confidence intervals to improve forecast accuracy

## Result

* Successfully analyzed COVID-19 case trends over time
* Identified patterns in case increases and decreases
* Generated a 30-day forecast of new cases
* Evaluated changes in case growth using month-over-month analysis
* Demonstrated practical skills in time-series analysis and forecasting

## Tools Used

* Power BI
* Power Query
* DAX
* Time-Series Forecasting
* Data Visualization
